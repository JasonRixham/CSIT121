Some More Takeaways

  • Organization on a web page will make it much easier to create and troubleshoot.
  • I'm certainly glad I don't program HTML for a living; it would be a fun hobby.
  • Resources for HTML are abundant; it is probably one of the most established languages in computer history.

New stuff learned since doing Lab 03

  • HTML is an boring document without CSS.
  • HTML so far, is very much easier to use than other languages I've learned.
  • CSS is quite a lot of fun, like painting a webpage.

Lab02 Reflections/Blog

Learning HTML was like gaining a superpower or casting magic; it allowed me to organize information, create links between pages, and present content on websites with confidence. The experience of learning this language introduced me to an exciting realm where creativity meets technology, giving me the ability to bring my ideas to life online.

Lab03 Reflections/Blog

When I started this assignment, I found HTML to be the foundational element of a webpage, similar to constructing a car frame. When I explored CSS, I realized my webpages could have pretty and vibrant colors and visually appealing aesthetics, much like the paint and interior of a car. HTML is the structure, while CSS adds the paint and glitter to bedazzle your webpage. This combination elevates a simple document into an engaging and attractive interface.
